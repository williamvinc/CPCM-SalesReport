## Deployment

Kamu bisa langsung menjalankan aplikasi ini menggunakan Docker Compose. Pastikan Docker sudah terinstall, lalu jalankan perintah berikut:

```bash
docker compose up --build -d
```

Aplikasi dapat diakses melalui browser di `http://110.239.80.161:8503` (atau IP/domain server Anda saat ini).
